# 说明

**项目描述：** 文档收留地。

**拉取git:**             `git clone https://gitee.com/guoshunfa/gsf-blog.git`

**加载文件包:**      `npm install`

**运行项目:**          `yarn dev`

**打包:**                 `yarn build`

**打包发布到Github Pages:**     `yarn deploy`

**配置:**          `/docs/.vuepress/config.js`

[**git地址**](https://gitee.com/guoshunfa/gsf-blog)
