# pandacode.cn

- [效果：熊猫 code | doc](https://pandacode.cn)

# 介绍

[熊猫 code](https://pandacode.cn)主要从Java出发进行学习、探索、记录。

# 快速上手

```shell
# 克隆项目到本地
git clone https://gitee.com/guoshunfa/panda-doc.git

# 进入目录
cd panda-doc

# 下载外部包
yarn install

# 运行项目
yarn start
```

# 其他操作

```shell
# 将网站内容推送至baidu
yarn baiduPush
```
