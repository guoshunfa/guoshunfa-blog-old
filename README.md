# pandacode.cn

- [效果：熊猫 code | doc](http://doc.pandacode.cn)

# 介绍

这个项目是[pandacode](https://pandacode.cn)对应[知识库](http://doc.pandacode.cn)平台，主要从Java出发进行学习、探索、记录。

# 快速上手

```shell
# 克隆项目到本地
git clone https://github.com/guoshunfa/pandacode-doc.git

# 进入目录
cd pandacode-doc

# 下载外部包
yarn install

# 运行项目
yarn dev
```

# 其他操作
```shell
# 打包发布到Github Pages
yarn deploy

# 将网站内容推送至baidu
yarn baiduPush
```
