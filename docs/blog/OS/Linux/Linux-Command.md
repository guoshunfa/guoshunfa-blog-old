# Linux 常用命令记录



| 命令         | 描述                                                         | 扩展描述                                                     |
| ------------ | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **ifconfig** | 查看端口                                                     |                                                              |
| **sudo**     | 执行命令后，输入当前管理员用户的密码就可以短暂得到超级用户的权限了。 | **sudo -i** ：通过此命令直接输入当前管理员用户的密码就可以进入root用户了。 |
