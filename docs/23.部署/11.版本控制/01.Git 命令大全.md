# Git 命令大全

## git更换远程仓库地址

```sh
git remote -v  #查看远端地址
git remote #查看远端仓库名
git remote set-url origin https://gitee.com/xx/xx.git (新地址)
```

## git 代码拉取

```sh
# 指定分支
$ git clone -b 开发分支 https://gitee.com/xx/xx.git
```
