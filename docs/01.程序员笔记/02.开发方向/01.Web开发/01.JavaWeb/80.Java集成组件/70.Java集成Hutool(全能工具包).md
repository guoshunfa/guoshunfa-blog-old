# Java集成 Hutool(全能工具包)

> [Hutool 官方文档](https://hutool.cn/docs/#/)

## 1. Maven

```xml
<dependency>
    <groupId>cn.hutool</groupId>
    <artifactId>hutool-all</artifactId>
    <version>5.7.6</version>
</dependency>
```
